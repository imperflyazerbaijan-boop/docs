LVINAI

LVINAI is a local desktop automation tool that helps the owner generate videos and upload them to YouTube and TikTok using official APIs and user authorization.

Important note (personal use)

This application is created and used only by the developer for their own accounts. It is not offered to the public, does not have external users, and does not provide a hosted service.

How it works

  1. User (the owner) runs LVINAI on their computer.
  2. LVINAI generates script/metadata, downloads media, and renders videos locally.
  3. User authorizes YouTube and TikTok once via OAuth (consent screen).
  4. LVINAI uploads videos via official APIs to the authorized account.

Data & security

  • Tokens are stored locally on the user's machine.
  • No user data is sold or shared with third parties.
  • User can delete local token files at any time to revoke access from this app.
