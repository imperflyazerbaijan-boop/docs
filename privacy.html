Privacy Policy — LVINAI

Last updated: 2025-12-21

Personal-use notice: LVINAI is a local desktop tool used only by its developer for their own accounts. It is not a public product and has no external users.

1. What LVINAI is

LVINAI is a desktop automation application that generates and uploads video content to platforms such as YouTube and TikTok, using official APIs and OAuth authorization.

2. Data we collect

LVINAI does not operate a hosted server and does not collect data to a central database. All operations are local.

  • OAuth tokens: access tokens / refresh tokens returned by YouTube/TikTok OAuth flows.
  • Local settings: configuration files stored on the user’s computer (e.g. API keys, preferences).
  • Generated media: videos/audio/images created and stored locally.

3. How data is used

  • OAuth tokens are used strictly to upload videos to the authorized account.
  • Settings and media files are used to generate and render videos locally.

4. Where data is stored

Data is stored locally on the user’s device. Example local files may include: youtube_token.json, tiktok_token.json, output_videos/.

5. Data sharing

LVINAI does not sell, rent, or share personal data with third parties. Data is only sent to the relevant platform (YouTube/TikTok) when the user chooses to upload content via official APIs.

6. Security

  • Tokens are stored locally on disk and are not transmitted to any third-party server controlled by LVINAI.
  • User should protect their computer account and files.

7. How to revoke access / delete data

  • Delete local token files (e.g. tiktok_token.json, youtube_token.json) to remove authorization from this app.
  • You may also revoke access from the platform’s security/account settings (recommended).
  • Delete local folders (e.g. output_videos/) to remove generated media.

8. Contact

If you have questions related to this policy during review, contact via the TikTok Developer Portal support thread for this app.
